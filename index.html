Sincronización entre dispositivos

Guarda el progreso (y el banco) como progreso.json en tu repositorio de GitHub y cárgalo desde cualquier dispositivo. Necesitas un token de acceso limitado a ese repositorio. La configuración se guarda solo en este navegador.

Cargar preguntas

Sube un archivo .json o pega el contenido abajo. Las preguntas nuevas se añaden al banco; las repetidas (mismo enunciado) se ignoran.

Formato esperado

Lista JSON de objetos. correcta es el índice de la opción correcta empezando en 0. dificultad admite facil, media o dificil (si falta, se asume media). explicacion es opcional pero muy recomendable.

[
  {
    "tema": "Tema 4 — Seguridad ciudadana",
    "pregunta": "La LO 4/2015 entró en vigor el...",
    "opciones": ["1 de enero de 2015", "1 de julio de 2015", "1 de abril de 2015"],
    "correcta": 1,
    "dificultad": "media",
    "explicacion": "Disposición final quinta: entrada en vigor el 1 de julio de 2015."
  }
]
